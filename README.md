# Playwright MCP

Python 3.12 project setup.

## Setup

### 1. Install Python 3.12 (if not already installed)

Using pyenv:
```bash
pyenv install 3.12.9
```

### 2. Create and activate virtual environment

```bash
# Create virtual environment
python3.12 -m venv venv

# Activate virtual environment
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install --upgrade pip
pip install -r requirements.txt
```

## Usage

After activating the virtual environment, you can:
- Run Python scripts: `python script.py`
- Install packages: `pip install <package-name>`
- Add packages to requirements.txt: `pip freeze > requirements.txt`

## Deactivate virtual environment

```bash
deactivate
```
